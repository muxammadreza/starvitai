# External standards referenced by the agent pack

This agent pack encodes best practices aligned to these well-known standards and primary sources.

## Security
- OWASP ASVS (Application Security Verification Standard)
- OWASP Top 10 for LLM Applications (prompt injection, excessive agency, etc.)
- SLSA (Supply-chain Levels for Software Artifacts)

## Reliability / Ops
- Google SRE Book (postmortem culture, SLO/error budget concepts)
- DORA metrics (four keys)

## Observability
- OpenTelemetry concepts + semantic conventions

## Accessibility
- W3C WCAG 2.2

Keep this list updated when rules/workflows add new standards.
