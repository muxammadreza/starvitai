---
description: Triage a potential safety signal
---

## Steps
1) Capture the signal:
   - who/what/when
   - affected cohort size
2) Classify severity:
   - informational / warning / urgent
3) Determine immediate action:
   - clinician notification
   - pause feature/model
   - escalate incident response
4) Start audit trail:
   - link to logs/metrics
   - record decision
5) Create follow-up tasks:
   - root cause
   - mitigation

## Output
Safety triage record + action items.
