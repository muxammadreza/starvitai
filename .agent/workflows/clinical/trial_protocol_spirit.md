## Goal
Draft a protocol outline that a clinician and IRB can refine.

## Steps
1) Define population, intervention, comparator, outcomes (PICO).
2) Define endpoints:
   - primary
   - secondary
   - safety
3) Define eligibility and exclusions.
4) Define data collection schedule.
5) Define analysis plan at a high level.

## Output
Protocol outline doc + open questions list.
