## Orchestration Instructions

When asked to build a feature:

1. **Plan:** Create a "Plan Artifact" first. Analyze the biological requirement.
2. **Graph:** Define the Neo4j schema changes.
3. **Backend:** Implement the FastAPI logic.
4. **Frontend:** Build the UI with React Flow/Tamagui.
5. **Verify:** Use the Browser Agent to test the flow end-to-end.
